最近 AI 圈子层出不穷出新东西，满屏都是 clawdbot，刷久了只感觉 FOMO 和焦虑。

今天想跟大家分享点打破信息茧房、刷新推荐算法的好玩东西。

＿

早上在 Reddit 上刷到一个超有趣的项目，开发者 @Waynedevvv 做了一个叫 PixelHQ 的 app：一个像素风格的办公室，里面的像素打工人会根据你电脑上的 Claude Code 状态动起来。

我更愿意戏称它为「 Claude Code 版的打工模拟器」。

打开 app 的那一瞬间，感觉回到了小时候玩口袋妖怪绿宝石的日子。画风一模一样，只不过里面跑的像素小人，从宝可梦变成了一个个为你打工的 AI 牛马 Agents。

＿

▌它能干嘛？

这些像素小人在你的 Claude Code 改代码时也会坐在桌前敲代码，当 Claude Code 思考时会走向白板假装思考，任务完成时还会起身庆祝（活脱脱的打工模范代表）。

技术上也很简单：在电脑上跑一个 CLI 进程，手机 app 通过局域网链接，Claude Code 的事件状态就会实时同步到手机上，触发对应的动画。

＿

▌原帖的评论区有不少人说：这东西好像没什么用啊。

但我觉得特别有意思！因为它把一个如此枯燥的事情变得有趣了起来。

Claude Code 为什么让人感觉很难上手？除了本身需要一点点编程知识，更多是因为它那生人勿近的终端界面。

一个黑框框，全是文字，没有按钮与说明，对很多非技术背景的朋友来说看起来是挺唬人的。

但这个无用的小项目，让 Vibe Coding 真的有了 vibe，有了氛围感。

而且说实话，复古的 CLI 虽然是 Agent 的最佳运行环境，但一定不是最直观的人机交互界面。我最近也在探索一些新的可能性，之后分享给大家。

＿

▌「无用」的价值

最近看起来有用，能够提高效率的 AI 工具确实越来越多了，但我觉得，有时候「没用」的东西，在这个时代反而更有价值。

当所有 AI 产品都在卷效率、卷功能的时候，能让人会心一笑的东西反而稀缺了。

C 端产品的生死，很多时候取决于 vibe，而不是做了多少功能，涵盖了多广的场景。

想起之前分享过的没有镜头的 AI 相机，它们都「没什么用」，却对我做 AI 产品有很多启发。

＿

借用 frog 公司我很喜欢的设计哲学结尾：Form follows Emotion 好的设计追随情感，不只是功能。
